# shc_navigator

# A webapp to show floor plan for SHC.

Install rimraf
> npm install rimraf

Run npm install to setup all the necessary library
> npm install

### index.html
* experiment with p5
* require p5.js and related libs
* [P5](https://p5js.org/)
	
### index2.html (old)
* experient with D3
* require d3.min.js (v4) and underscore-min.js
* [D3](https://d3js.org/)
* [Underscore](http://underscorejs.org/)
