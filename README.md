# shc_navigator

# A webapp to show floor plan for SHC.

Install node.js / npm
https://www.npmjs.com/get-npm


Install rimraf
> npm install rimraf

Run npm install to setup all the necessary libraries
> npm install

Start web service
> gulp

The homepage will be available in http://localhost:8001

### index.html
* experiment with p5
* require p5.js and related libs
* [P5](https://p5js.org/)
	
### index2.html (old)
